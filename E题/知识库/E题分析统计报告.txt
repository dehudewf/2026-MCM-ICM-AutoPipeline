【E题典型章节结构】(基于O奖论文统计)
======================================================================

1. Summary/Abstract (摘要)
2. Contents (目录)
3. Introduction
   3.1 Problem Background
   3.2 Problem Restatement / Our Work
4. Assumptions & Notations
5. Model I: [核心计算模型]
   5.1 Model Establishment
   5.2 Solution
6. Model II: [价值评估模型]
   6.1 Indicator System
   6.2 Weight Determination (AHP/EWM)
   6.3 Evaluation Method (TOPSIS/GE)
7. Model III: [决策模型]
   7.1 Decision Framework
   7.2 Transition Point Analysis
8. Case Study / Application
   8.1 Application to [具体案例]
   8.2 Results Analysis
9. Sensitivity Analysis (敏感性分析) ★必需
10. Strengths and Weaknesses (优缺点) ★必需
11. Conclusion (结论)
12. References (参考文献)
13. Appendix
    - Code
    - Additional Data
    - Newspaper Article (E题特有)

【E题敏感性分析对象】(不同于C题)
======================================================================
✅ E题分析对象：
- 砍伐周期 (cutting cycle)
- 砍伐强度 (cutting intensity)
- 权重参数 (weights in AHP)
- 风险阈值 (risk thresholds)
- 转换点条件 (transition point conditions)
- 贴近度参数 (TOPSIS parameters)

❌ 不分析：
- 模型超参数（学习率、深度等）
- 数据扰动
- 训练集/测试集划分
