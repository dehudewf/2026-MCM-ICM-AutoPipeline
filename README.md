# MCM/ICM C题论文分析工具集

## 📚 项目简介

本项目是一个专门用于分析 MCM/ICM（美国大学生数学建模竞赛）C题优秀论文的智能工具集。通过自动化分析和人工审核相结合的方式，帮助参赛者快速学习历年优秀论文的解题思路、模型选择和写作技巧。

## 🎯 主要功能

### 1. 智能论文分析
- 📄 自动提取PDF论文内容
- 🤖 智能识别使用的数学模型
- 📊 统计图表数量
- 📝 提取论文结构和章节
- 💡 生成分析报告

### 2. 数据统计与可视化
- 📈 模型使用频率统计
- 📉 年度趋势分析
- 🏆 高频模型排行
- 📋 详细统计报告

### 3. 知识库构建
- 📚 模型知识库
- 🔍 论文检索系统
- 💾 结构化数据存储

## 🚀 快速开始

### 环境要求

- Python 3.7+
- 必需的库：
  ```bash
  pip install pandas openpyxl PyPDF2
  ```

### 使用步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/dehudewf/MCM-ICM_C_Repository.git
   cd MCM-ICM_C_Repository
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **运行分析脚本**
   ```bash
   python 论文分析助手_优化版.py
   ```

4. **查看结果**
   - `论文分析结果_优化版.xlsx` - 详细分析结果
   - `分析统计报告.txt` - 统计汇总
   - `C题模型知识库_优化版.xlsx` - 模型使用统计

## 📊 数据说明

### 2024年C题：网球动量分析

**题目：** Momentum in Tennis  
**数据文件：**
- `Wimbledon_featured_matches.csv` - 温布尔登比赛逐分数据
- `data_dictionary.csv` - 数据字典（43个变量）

**核心问题：**
1. 如何量化网球比赛中的"动量"？
2. 动量如何影响比赛结果？
3. 能否预测动量变化？
4. 如何应用于实际比赛策略？

## 🔧 功能详解

### 论文分析助手

**自动提取内容：**
- ✅ 论文页数
- ✅ 摘要内容
- ✅ 使用的模型（基于关键词匹配）
- ✅ 章节结构
- ✅ 图表数量

**需人工补充：**
- 📝 核心模型（确认自动识别结果）
- 💡 创新点
- 📊 数据来源
- ✔️ 验证方法
- ⭐ 可借鉴度评级

### 分析框架

提供系统化的论文分析方法：
1. **快速分析检查清单**（5-10分钟/篇）
2. **详细分析模板**（包含问题分析、模型体系、技术亮点等）
3. **跨论文对比分析**
4. **解题范式总结**

## 🎓 学习建议

### 第一轮：快速浏览（2-3天）
- 目标：每篇30分钟，重点看摘要、图表、结论
- 任务：填写分析记录表

### 第二轮：精读优秀论文（3-4天）
- 目标：选择3-5篇最优论文深入学习
- 任务：详细分析模型、复现算法

### 第三轮：总结提炼（1-2天）
- 目标：归纳解题范式
- 任务：完成自己的解题模板和模型库

## ⚠️ 注意事项

1. **自动识别准确性**
   - PDF文本提取质量受原文件影响
   - 模型识别基于关键词匹配，可能存在误判
   - 建议人工复核所有自动识别结果

2. **数据完整性**
   - 部分PDF可能无法正常提取
   - 图表统计可能存在偏差
   - 创新点需要人工提炼

3. **版权说明**
   - 本项目仅用于学习研究
   - 论文版权归原作者所有
   - 请勿用于商业用途
