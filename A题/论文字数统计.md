## Notes

### Data Source
- **Papers analyzed**: 5 A-problem O-award papers from `pre-theis-eng` folder
- **Years covered**: 2022, 2023, 2024, 2025
- **Selection criteria**: Representative papers from different years showing consistent quality standards

### A-Problem Characteristics
- **Core focus**: Continuous modeling (ODE/PDE systems), numerical simulation, optimization algorithms
- **Model structure**: Typically 3 interconnected models
- **Average page count**: 20-25 pages (excluding appendix)
- **Typical word count**: 9,132 ± 800 words for main body

### Section-Specific Observations

**Summary/Abstract (350 words, 3.8%)**
- Concise problem statement + 2-3 key models + quantitative results
- Typically 1 paragraph, no figures

**Introduction (712 words, 7.8%)**
- Problem Background: Real-world context with data citations (~314 words)
- Problem Restatement: Clear task decomposition (~260 words)
- Our Work: Brief methodology overview with flowchart (~138 words)

**Assumptions (438 words, 4.8%)**
- A-problem papers typically have 5-8 assumptions
- Each assumption requires explicit justification
- Emphasis on physical constraints and mathematical validity

**Model Development (4520 words, 49.5%)**
- **Largest section**: Constitutes half of the paper
- Model I: Core continuous-time model (ODE/PDE formulation)
- Model II: Extensions (interaction terms, thermal coupling, etc.)
- Model III: Numerical methods and solution algorithms
- Heavy use of equations, derivations, and theoretical analysis

**Results (1316 words, 14.4%)**
- Quantitative validation of predictions
- Comparison with observed/published data
- Multiple scenarios analyzed

**Sensitivity Analysis (658 words, 7.2%)**
- **Required section** for A-problem
- Parameter variation analysis
- Robustness testing under assumption changes

**Strengths & Weaknesses (352 words, 3.9%)**
- Balanced evaluation (typically 3 strengths, 2 weaknesses)
- Honest limitations
- Future improvement directions

### Quality Benchmarks (O-Award Standards)
- **Abstract**: Must include quantitative results (e.g., "MAPE < 12%", "TTE ± 0.5h")
- **Model clarity**: Each equation numbered and explained
- **Data rigor**: All parameters justified with literature or measurements
- **Innovation metric**: Must demonstrate improvement over baseline (e.g., "R² = 0.89 vs. baseline 0.65")
- **Figure quality**: 300+ DPI, publication-ready, self-explanatory captions
