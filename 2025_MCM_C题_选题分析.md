## 三、核心子问题建模方案

### 问题1.1: 奖牌数预测模型开发

**建模思路：**
```
Step 1: 数据预处理
  - 缺失值填充（线性插值/KNN）
  - 异常值检测（Z-score/IQR）
  - 特征标准化（Min-Max/Z-score）

Step 2: 特征工程
  - 滞后特征：past_1, past_2, past_3届奖牌数
  - 滚动统计：3届/5届移动平均
  - 趋势特征：增长率、加速度
  - 交互特征：GDP × 人口、投资 × 项目数

Step 3: 模型训练
  - 训练集：1896-2016（80%）
  - 验证集：2020（10%）
  - 测试集：2024（10%）

Step 4: 模型融合
  Ensemble = 0.3×LSTM + 0.4×XGBoost + 0.3×Prophet
```

**评估指标：**
- MAE (Mean Absolute Error)
- RMSE (Root Mean Square Error)
- MAPE (Mean Absolute Percentage Error)
- R² Score

### 问题1.2: 2028年预测

**预测流程：**
1. **基础预测：** 使用训练好的模型预测各国基础奖牌数
2. **主办国调整：** 美国作为主办国，历史数据显示平均增幅15-25%
3. **项目变化调整：** 2028年新增/删除项目的影响
4. **不确定性量化：** 95%置信区间

**预测输出示例：**

| 国家 | 预测金牌 | 95%CI | 预测总奖牌 | 95%CI |
|------|----------|-------|------------|-------|
| 美国 | 48-52    | [45,55] | 135-145  | [130,150] |
| 中国 | 38-42    | [35,45] | 85-95    | [80,100] |
| 日本 | 18-22    | [15,25] | 42-48    | [38,52] |

### 问题1.3: 南美国家获奖牌预测

**分析维度：**
1. **历史表现：** 巴西、阿根廷、哥伦比亚等国历史数据
2. **经济发展：** GDP增长趋势
3. **体育投资：** 政府体育预算变化
4. **优势项目：** 足球、排球、柔道等

**预测方法：**
- 使用分层模型：先预测区域总奖牌数，再分配到各国
- 考虑区域特殊性：南美国家在团体项目上的优势

### 问题1.4: 比赛数量与奖牌关系

**建模方法：**
1. **相关性分析：** Pearson/Spearman相关系数
2. **回归分析：** 
   ```
   Medals_i = β₀ + β₁×Events_i + β₂×Athletes_i + β₃×GDP_i + ε
   ```
3. **因果推断：** 使用工具变量法（IV）或倾向得分匹配（PSM）

**可视化：**
- 散点图：比赛数量 vs 奖牌数
- 热力图：不同体育项目对各国的贡献度

## 四、"优秀教练效应"分析

### 2.1 数据证据分析

**假设：** 优秀教练可以跨国流动，影响奖牌分布

**分析方法：**
1. **案例研究：** 
   - 中国乒乓球教练在其他国家的成功案例
   - 俄罗斯体操教练在美国的影响

2. **统计检验：**
   ```python
   # 双样本t检验
   H₀: μ(有优秀教练) = μ(无优秀教练)
   H₁: μ(有优秀教练) > μ(无优秀教练)
   ```

3. **回归模型：**
   ```
   Medal_gain = α + β×Coach_quality + γ×Controls + ε
   ```

### 2.2 三国案例选择

**选择标准：**
- 代表性：不同经济发展水平
- 数据可得性：教练流动数据完整
- 对比性：有/无优秀教练的对照

**推荐案例：**
1. **肯尼亚** - 长跑项目，教练体系成熟
2. **牙买加** - 短跑项目，博尔特时代的教练影响
3. **新加坡** - 乒乓球项目，引进中国教练

## 五、模型假设与验证

### 5.1 核心假设

**Assumption 1: 历史趋势延续性**
- 假设：各国体育发展遵循历史规律
- 验证：通过2020/2024年数据回测

**Assumption 2: 经济-奖牌正相关**
- 假设：GDP增长带动体育投资和奖牌数
- 验证：相关性分析 + 格兰杰因果检验

**Assumption 3: 主办国优势**
- 假设：主办国奖牌数平均增加15-25%
- 验证：历史主办国数据统计

**Assumption 4: 项目数量影响**
- 假设：参赛项目数与奖牌数正相关
- 验证：回归分析 + 控制变量

**Assumption 5: 教练效应可量化**
- 假设：优秀教练带来显著奖牌增长
- 验证：案例研究 + 统计检验

### 5.2 敏感性分析

**测试场景：**
1. GDP增长率 ±2%
2. 体育投资 ±10%
3. 参赛项目数 ±5项
4. 主办国效应 ±5%

**鲁棒性检验：**
- 交叉验证（K-fold CV）
- 留一法（Leave-One-Out）
- 蒙特卡洛模拟（10000次）
